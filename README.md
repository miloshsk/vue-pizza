# Установка через консоль
1. Клонировать репозиторий  ```git clone https://github.com/miloshsk/vue-pizza```
2. Перейти в созданную директорию ```cd vue-pizza```
3. Установить зависимости ```npm i```
4. Запустить ```npm run dev```
